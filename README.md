# change_girls plugin

[npm version](https://www.npmjs.com/package/gitbook-plugin-change_girls)

This plugin add backgroundImg,  and auto change these imgs.

```json
"plugins":["change_girls"]

"pluginsConfig": {
    
    "change_girls" : {
        time : 5,
        "urls" : [
            "girlUrl1", "girlUrl2",...
        ]
    }
}
```

> #### *instructions:*

- time , toggle time, second
- urls, url of images
